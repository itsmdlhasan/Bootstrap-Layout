I have fixed with container layout

So, max-width size of mine will change on different screen size as below:

  1. XXL(>/=1400px) = 1320px
  2. XL(>/=1200px) = 1140px
  3. LG(>/=992px) = 960px
  4. MD(>/=768px) = 720px
  5. SM(>/=576px) = 540px
  6. XSM(less than 576px) = 100%

I have fluid with container layout

So, I will take the full width of viewport

I have fixed with container layout with padding

I have a padding of 5, where 0=0, 1=.25rem, 2=0.5rem, 3=1rem, 4=1.5rem, 5=3rem and 1rem means 16px of the body root by default if not changed.
